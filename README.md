# C++ To-Do App

A simple command-line To-Do application written in C++.
Tasks are saved to a file and persist after closing the program.

## Features

- Add tasks
- Show all tasks
- Delete tasks
- Tasks are saved automatically after every action

## How to Compile

g++ todo.cpp -o todo

## How to Run

Linux / WSL:
./todo

Windows:
.\todo.exe

## How to Use

1. Start the program
2. Choose an option from the menu:
   1 - Add a new task
   2 - Show all tasks
   3 - Delete a task
   4 - Close the program
3. Tasks are saved automatically in todos2.txt

## Requirements

- C++ Compiler (g++)

### Windows
- MinGW or WSL: https://learn.microsoft.com/en-us/windows/wsl/install
- Git: https://git-scm.com/download/win

### Linux / WSL
sudo apt install g++
