# C++ To-Do App

A simple command-line To-Do application written in C++.
Tasks are saved to a file and persist after closing the program.

## Features

- Add tasks
- Show all tasks
- Delete tasks
- Tasks are saved automatically after every action

## How to Compile

\```bash
g++ todo.cpp -o todo
\```

## How to Run

\```bash
# Linux / WSL:
./todo

# Windows:
.\todo.exe
\```

## Requirements

\```bash
# Linux / WSL:
sudo apt install g++
\```
